An MCP server for Claude Code.

Automatically loads the right thinking for what you're working on.

INSTALL SOURCE

This assists your coding. It doesn't replace your judgment.
Skills and knowledge are templates. Edit or add your own.

WHAT HAPPENS

You: "Review this API endpoint" Crucible detects: .py → python, backend, api Crucible loads: backend-engineer + API_DESIGN.md + SECURITY.md Claude now thinks with: - Input validation patterns - Error handling conventions - N+1 query awareness - Auth/authz questions

Not a template dump. An active server that detects context and loads relevant patterns.
Optionally runs static analysis (semgrep, ruff, slither, bandit) and includes findings.

CUSTOMIZE

Override at any level:

1. Project: .crucible/skills/ ← Highest priority
2. User: ~/.claude/crucible/
3. Bundled: (package defaults) ← Lowest priority
# Copy a skill to customize crucible skills init security-engineer # → Creates .crucible/skills/security-engineer/SKILL.md # → Edit it for your project # Same for knowledge crucible knowledge init SECURITY # → Creates .crucible/knowledge/SECURITY.md

INSTALL

pip install crucible-mcp

Add to Claude Code:

{ "mcpServers": { "crucible": { "command": "crucible-mcp" } } }

With hot reload (recommended):

{ "mcpServers": { "crucible": { "command": "mcpmon", "args": ["--watch", "~/.crucible/", "--", "crucible-mcp"] } } }
